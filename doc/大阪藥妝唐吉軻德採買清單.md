## 💊 藥妝店挑選策略

### 推薦順序
1. **大國藥妝（Daikoku Drug）** — 平均最便宜
   - 心齋橋本店、難波、道頓堀都有大店
2. **松本清（Matsumoto Kiyoshi）** — 分店最多、優惠券多
3. **Welcia / Sundrug** — 第二梯次比較

### 免稅與折扣
- **免稅門檻**：同店單日 5,500 日圓含稅以上（消費稅 10% 退）
- **疊加優惠**：護照免稅 + LINE / 樂吃購優惠券 + 信用卡優惠（最高 34% off）
- 結帳前出示：護照 + 優惠券 QR code

### 行前下載
- [ ] 松本清優惠券（letsgojp、Funliday 都有 QR 連結）
- [ ] 大國藥妝優惠券
- [ ] 唐吉軻德 LINE 好友（7% 折扣券）
