## 📅 採買時程建議（已對齊 `20260516-0521 大阪6D5N.md` 行程）

### Day 1｜5/16（六）｜心齋橋夜逛 + 飯店附近 Donki
- **18:00–20:00 心齋橋逛街時順手**：
  - [ ] PABLO 半熟起司塔（心齋橋本店）
  - [ ] 大阪花餅乾（阿彌陀池大黒，心齋橋本店）
  - [ ] 食倒太郎布丁（道頓堀，心齋橋走 5 分）
- **21:00 Maruyoshi 谷四店晚餐後 → 飯店附近唐吉軻德分店**
  - [ ] 美妝、藥品、零食主力採購（清單見 `大阪藥妝唐吉軻德採買清單.md`）
  - [ ] 唐吉軻德 130L 行李袋（如需裝戰利品）

### Day 2｜5/17（日）｜神戶整天（無採買）
- 神戶松露已加 KIX 購物車，現場別重複買
- 元町商店街可看看，但別揹太多影響後續行程

### Day 3｜5/18（一）｜黑門 × 日本橋 × 新世界
- [ ] **黑門市場** — 海鮮乾貨、抹茶、茶等可長放食材
- [ ] **日本橋電器街** — 模型、動漫周邊（個人興趣）
- [ ] 黑門 / 日本橋附近 **大國藥妝**（道頓堀店或鄰近分店）補貨（單日 5,500 円免稅門檻）

### Day 4｜5/19（二）｜USJ 全日（無採買）
- 園區內限定紀念品另計

### Day 5｜5/20（三）｜🔥 梅田主力採買日（馬喰一代午餐後 13:30 開始，需於 16:30 前收手）

> 17:00 鴨蔥拉麵 → 19:00 藍天大廈，採買最晚 **16:30** 要結束

- **13:30 B1 — Drooly 蜂蜜起司餅乾**（預計排隊 20–30 分鐘，先到先排）
  - [ ] Drooly 排完順走 **Grand Calbee 厚切洋芋片**（阪急同棟）
- **14:30–16:00 百貨地下自由逛**
  - [ ] Henri Charpentier 試吃（KIX 已買，看有無限定口味再決定加購）
  - [ ] 月化粧 牛奶饅頭（青木松風庵專櫃）
- 16:15 前採買完成，輕鬆步行去吃鴨蔥拉麵

### Day 6｜5/21（四）｜返台 + KIX 取貨
- [ ] 09:45 大阪城散策 → 順路看看新大阪車站伴手禮
- [ ] 11:00 NewBabe 炸豬排
- [ ] **14:35 抵 KIX → 出境後到直營店取線上購物車 9 件**
- [ ] 順便看看「東京芭娜娜 巧克力蛋糕 關西國際機場限定」¥2,800 現場有沒有貨（線上缺貨）
- ⚠️ **PTT 大量警告**：關西機場改建中現場店少，別把希望留給機場逛
- ⏰ 預留 1h40：14:35 取貨完成 + Check-in + 安檢 + 出境，16:15 起飛
